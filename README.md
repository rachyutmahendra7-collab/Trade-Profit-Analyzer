# Trade Profit Analyzer

A simple C++ program that calculates profit or loss for multiple trades.

### Features
- Input buy/sell prices
- Calculates profit/loss automatically
- Displays "Great Trade!" or "Big Loss!" messages
- Shows total profit/loss

## How To Run
g++ $(find . -name "Trade-Profit-Analyzer.cpp") -o Trade-Profit-Analyzer && ./Trade-Profit-Analyzer
